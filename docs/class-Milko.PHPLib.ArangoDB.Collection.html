Class Collection

Collection ancestor object.

This concrete class is the implementation of a ArangoDB collection, it implements the inherited virtual interface to provide an object that can manage ArangoDB collections.

This class stores the triagens\ArangoDb\Collection object and instantiates at runtime the triagens\ArangoDb\CollectionHandler to perform operations.

ArrayObject implements IteratorAggregate, Traversable, ArrayAccess, Serializable, Countable

# insert( array|object $theRecord, boolean $doMany, mixed $theOptions = NULL )

Insert one or more records.

Insert one or more records.

This method should insert the provided record or records, the method expects the following parameters:

  • $theRecord: The record to be inserted.
  • $theOptions: An array of options representing driver native options.
  • $doMany: TRUE provided many records, FALSE provided one record.

This method assumes that the server is connected, it is the responsibility of the caller to ensure this.

This method must be implemented by derived concrete classes.
