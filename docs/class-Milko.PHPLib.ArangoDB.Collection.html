Class Collection

Collection ancestor object.

This concrete class is the implementation of a ArangoDB collection, it implements the inherited virtual interface to provide an object that can manage ArangoDB collections.

This class stores the triagens\ArangoDb\Collection object and instantiates at runtime the triagens\ArangoDb\CollectionHandler to perform operations.

ArrayObject implements IteratorAggregate, Traversable, ArrayAccess, Serializable, Countable

# normaliseSelectedDocument( Milko\PHPLib\Container $theDocument, mixed $theData )

Normalise selected document.

Normalise selected document.

This method will be called when a Container instance has been selected from the current collection via a query, its duty is to pass information back to the document, including eventual internal native database properties.

The method expects a single parameter which should be a Container instance.

The method is implemented in this class to handle Document instances:

  • Document::IsPersistent(): If the provided document is a Document instance, the document's persistent state will be set.
  • Document::IsModified(): If the provided document is a Document instance, the document's modification state will be reset.

In derived classes you should first manage internal database properties, if relevant, then call the current method.

The selected document.
