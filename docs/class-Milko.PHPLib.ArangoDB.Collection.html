Class Collection

Collection ancestor object.

This concrete class is the implementation of a ArangoDB collection, it implements the inherited virtual interface to provide an object that can manage ArangoDB collections.

This class stores the triagens\ArangoDb\Collection object and instantiates at runtime the triagens\ArangoDb\CollectionHandler to perform operations.

ArrayObject implements IteratorAggregate, Traversable, ArrayAccess, Serializable, Countable

# doDeleteByCursor( triagens\ArangoDb\Cursor $theCursor, array $theOptions )

Delete the the documents from a cursor.

Delete the the documents from a cursor.

This method will delete the first or all documents contained in the provided cursor, the method expects the following parameters:

  • $theCursor: The cursor.
  • $theOptions: An array of options:
    • kTOKEN_OPT_MANY: This option determines whether to delete only the first selected record or all:
      • TRUE: Delete the whole selection.
      • FALSE: Delete only the first selected record.

The method will return the number of deleted records.
