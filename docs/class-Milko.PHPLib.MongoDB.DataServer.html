Class DataServer

MongoDB data server object.

This concrete class is the implementation of a MongoDB data server, it implements the inherited virtual interface to provide an object that can manage MongoDB databases, collections and documents.

ArrayObject implements IteratorAggregate, Traversable, ArrayAccess, Serializable, Countable
