Class Database

Database ancestor object.

This abstract class is the ancestor of all classes representing database instances.

This class features a data member that holds the Milko\PHPLib\DataServer object that instantiated the current database and another data member that holds the native driver's database object.

The class implements a public interface that deploys the common interface of derived concrete classes:

  • Milko\PHPLib\Server: Return the database server object.
  • Milko\PHPLib\Database::Connection(): Return the database native driver object.
  • Milko\PHPLib\Database::Drop(): Drop the database; this method is virtual.
  • Milko\PHPLib\Database::ListCollections(): Return the list of collection names on the database.
  • Milko\PHPLib\Database::WorkingCollections(): Return the list of working collection names.
  • Milko\PHPLib\Database::RetrieveCollection(): Create or retrieve a collection object.
  • Milko\PHPLib\Database::ForgetCollection(): Clear or dispose of a collection object.
  • Milko\PHPLib\Database::EmptyCollection(): Empty a collection.
  • Milko\PHPLib\Database::DropCollection(): Drop a collection.

The object's inherited array will be populated with collections as they are used, the offsets represent the collection name and the values will be Milko\PHPLib\Collection objects.

Most of the above methods feature a bitfield parameter that determines the actions to take in the event a collection is not found:

  • kFLAG_CREATE: If set and the indicated collection was not found, it will be created in the process.
  • kFLAG_ASSERT: If set, an exception will be raised if the indicated collection cannot be found and the kFLAG_CREATE flag is not set.

The public methods do not implement the actual operations, this is delegated to a protected virtual interface which must be implemented by derived concrete classes:

  • Milko\PHPLib\Database::databaseNew(): Instantiate a driver native database instance.
  • Milko\PHPLib\Database::databaseName(): Return the database name.
  • Milko\PHPLib\Database::collectionList(): Return the list of database collection names.
  • Milko\PHPLib\Database::collectionCreate(): Create and return a Milko\PHPLib\Collection object corresponding to the provided name.
  • Milko\PHPLib\Database::collectionRetrieve(): Create and/or return a Milko\PHPLib\Collection object corresponding to the provided name.

Each time a collection object is retrieved, the object will store it in the mCollections data member for fast subsequent retrievals.

public Milko\PHPLib\Collection
# RetrieveCollection( string $theCollection, string $theFlags = Server::kFLAG_DEFAULT, array $theOptions = NULL )

Return a collection object.

Return a collection object.

This method can be used to create or retrieve a collection object, it features the following parameters:

  • $theCollection: The collection name.
  • $theFlags: A bitfield providing the following options:
    • kFLAG_CREATE: If set, the method will create the collection if it doesn't already exist.
    • kFLAG_ASSERT: If set, the method will raise an exception if the collection cannot be found.
  • $theOptions: An array of options representing driver native options.

The method will either return a Milko\PHPLib\Collection object, or NULL if the collection was not found.

It is the responsibility of the caller to ensure the server is connected.

Return a native database object.
