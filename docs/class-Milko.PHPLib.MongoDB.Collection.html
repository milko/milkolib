Class Collection

Collection ancestor object.

This concrete class is the implementation of a MongoDB collection, it implements the inherited virtual interface to provide an object that can manage MongoDB collections.

ArrayObject implements IteratorAggregate, Traversable, ArrayAccess, Serializable, Countable
