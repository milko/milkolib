Class Collection

Collection ancestor object.

This abstract class is the ancestor of all classes representing collection instances.

This class features a data member that holds the Milko\PHPLib\Database object that instantiated the current collection and another data member that holds the native driver's collection object.

The class implements a public interface that deploys the common interface of derived concrete classes:

  • Milko\PHPLib\Server: Return the database server object.
  • Milko\PHPLib\Database: Return the database object.
  • Milko\PHPLib\Collection::Connection(): Return the collection native driver object.
  • Milko\PHPLib\Collection::InsertOne(): Insert one record.
  • Milko\PHPLib\Collection::InsertMany(): Insert many records.
  • Milko\PHPLib\Collection::UpdateOne(): Update one record.
  • Milko\PHPLib\Collection::UpdateMany(): Update many records.
  • Milko\PHPLib\Collection::ReplaceOne(): Replace one record.
  • Milko\PHPLib\Collection::FindOne(): Find one record.
  • Milko\PHPLib\Collection::FindMany(): Find many records.
  • Milko\PHPLib\Collection::QueryCollection(): Perform a query.
  • Milko\PHPLib\Collection::DeleteOne(): Delete one record.
  • Milko\PHPLib\Collection::DeleteMany(): Delete many records.

The public methods do not implement the actual operations, this is delegated to a protected virtual interface which must be implemented by derived concrete classes:

  • Milko\PHPLib\Collection::newCollection(): Instantiate a driver native database instance.
  • Milko\PHPLib\Collection::collectionName(): Return the collection name.
  • Milko\PHPLib\Collection::insert(): Insert one or many records.
  • Milko\PHPLib\Collection::update(): Update one or many records.
  • Milko\PHPLib\Collection::replace(): Replace one or many records.
  • Milko\PHPLib\Collection::find(): Find one or many records.
  • Milko\PHPLib\Collection::query(): Query records.
  • Milko\PHPLib\Collection::delete(): Delete one or many records.
