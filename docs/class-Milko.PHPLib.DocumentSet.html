Class DocumentSet

Document set object.

This class implements a set of documents that can be inserted into a collection.

The class is instantiated by providing the collection and the buffer size: as we add documents to the set, when the buffer count is reached the documents will be inserted.

The destructor will take care of inserting the pending buffer contents.
