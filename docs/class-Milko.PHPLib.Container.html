Class Container

Container base object.

This class is the ancestor of classes that that handle structured data and that can be persistent.

The attributes of the object represent transient information which is private to the object itself, this data is stored in the object's data members and is not considered by the persistent framework of this library.

The properties of the object represent the persistent information carried by the object, this data is stored in the array data member inherited by the ArrayObject ancestor class and are accessed through the array management syntax, this data is used by the persistence framework of the library.

Properties cannot hold the NULL value, setting a property to that value will result in that property being deleted.

The class implements an interface that standardises the way attributes and properties are managed:

  • Attributes: a protected interface can be used to standardise the behaviour of member accessor methods, in general there should be a single public method for a specific attribute that will store, retrieve and delete attributes, depending on the provided value:
    • NULL: Retrieve the attribute value.
    • FALSE: Reset the attribute value to NULL.
    • other: Any other type will result in the attribute being set to that value.
  • Properties: a public interface will take care of implementing the standard behaviour, this to ensure no warnings are issued:
    • Setting a property to NULL will delete the property.
    • Retrieving a property that does not exist will return the NULL value.
    • Deleting a property that does not exist will do nothing.

# manageAttribute( mixed & $theMember, mixed $theValue = NULL )

Manage an attribute

Manage an attribute

This library implements a standard interface for managing attributes using accessor methods, attributes are stored in the object's data members, this method implements this interface:

  • &$theMember: Reference to the object property being managed.
  • $theValue: The attribute value or operation:
    • NULL: Return the current attribute value.
    • FALSE: Reset the attribute to NULL and return the old value.
    • other: Set the attribute with the provided value and return it.

Parameters

$theMember
$theMember Reference to the data member.
$theValue
Value or operation.

Returns

mixed
Old or current attribute value.

Example

$this->manageAttribute( $member, "value" ); // Will set a value in that member.
$this->manageAttribute( $member, NULL ); // Will return the member's current value.
$this->manageAttribute( $member, FALSE ); // Will set the member's value to NULL.

# manageProperty( mixed $theProperty, mixed $theValue = NULL )

Manage a property

Manage a property

Properties in this class are stored in an array and accessed by offset, besides this method, dedicated accessor methods can be used: this method should be used to provide a consistent interface when deploying member accessor methods that manage properties.

The method accepts the following parameters:

  • $theProperty: Property offset.
  • $theValue: The property value or operation:
    • NULL: Return the current property value.
    • FALSE: Delete the property and return the old value.
    • other: Set the property with the provided value and return it.

Parameters

$theProperty
Property offset.
$theValue
Value or operation.

Returns

mixed
Old or current property value.

Example

$this->manageProperty( $offset, "value" ); // Will set a value in that offset.
$this->manageProperty( $offset, NULL ); // Will return the value at that offset.
$this->manageProperty( $offset, FALSE ); // Will delete that offset.
